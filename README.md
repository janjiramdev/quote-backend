# quote-backend

## Description

This project is quote backend services with TypeScript, NestJS and MongoDB

## Prerequisite

- Node v20.x.x or later
- Docker (MongoDB v8.0)

## Project setup

- Setup database

```bash
$ docker-compose up -d
```

- Config env follow up .env.example
- Install NodeJS dependencies

```bash
$ npm install
```

## Compile and run the project

```bash
$ npm run start:dev
```
